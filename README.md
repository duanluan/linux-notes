## 📖 项目简介

本仓库记录 Linux（主要是 Manjaro / Arch 系）使用过程中的：
- ⚙️ 环境配置与系统调优
- 🗃️ 常用软件与命令速查
- 🧰 开发工具链（Shell / 包管理 / 构建）
- 🌐 网络 & 代理 & 远程管理
- 🔐 安全与权限实践
- 🪛 常见故障排查（坑点 & 修复）
- 🖥️ 图形界面 / KDE / 字体渲染优化
- 📝 其它零碎但超实用的 Tips

> 目标定位：一个“随时查”、能“快速复制使用”的笔记型知识库。

## 🤝 如何贡献

欢迎各种形式的贡献（Star / Issue / PR）：

1. ⭐ 点 Star（第一步最简单也最重要）
2. 🐞 提 Issue：发现错误或希望新增主题
3. 🛠 提 PR：在 `docs/` 下新增或修改文档
4. 🗣 讨论：可以在 Issue 中提出你的环境差异
5. 🔍 保持风格：内容偏“实用 + 可复制”，避免空谈

### 提交建议
- 使用简洁标题
- 命令行加注释（不要删除已有注释）✅
- 多用示例 vs 只描述
